# Software Engineering Portfolio - 2026

A modern, responsive portfolio website built with the 2026 tech stack standards, showcasing software engineering projects and technical skills.

## 🚀 Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn/UI
- **Icons**: Lucide React
- **Deployment**: Vercel (recommended)

## ✨ Features

- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **Projects Showcase**: 4 featured projects with live demos and source code links
- **Technical Stack**: Visual representation of skills and proficiency levels
- **Case Study**: Deep dive into the most challenging project with problem-solving insights
- **Contact Section**: Professional contact information and resume download
- **AI Integration**: Showcases experience with modern AI APIs
- **Mobile Responsive**: Optimized for all device sizes
- **SEO Optimized**: Meta tags, Open Graph, and structured data
- **Performance**: Optimized for 90+ Lighthouse scores

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/mshoaibmalik/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🔧 Customization

### Personal Information
Update the following files with your information:

1. **Hero.tsx**: Change name, tagline, and contact links
2. **Projects.tsx**: Replace with your actual projects
3. **TechStack.tsx**: Update with your technical skills
4. **CaseStudy.tsx**: Replace with your most challenging project
5. **Contact.tsx**: Update contact details and social links
6. **layout.tsx**: Update metadata and SEO information

### Adding Your Resume
1. Place your PDF resume in the `public/` folder
2. Update the filename in `Contact.tsx` if needed

### Project Requirements
Each project should include:
- Live demo URL
- GitHub repository URL
- Technology stack badges
- Brief description
- Featured project designation
